## 📋 Project Overview

**TIXORA** is an event ticket booking platform that allows users to browse events, purchase tickets, and display QR code tickets for entry. It serves as a complete event management solution with a clean, modern interface built with **React**.

## ✨ Features

### User Features
- **🔍 Browse Events** — Discover upcoming events with details like date, venue, and pricing
- **🎟️ Buy Tickets** — Purchase tickets for events securely
- **📱 QR Code Tickets** — Generate and display scannable QR codes for entry verification
- **📊 Event Dashboard** — View your purchased tickets and event history
- **👤 User Profile** — Manage your account and preferences
- **🔐 Authentication** — Register and login to your account

### Technical Features
- ✅ React Components
- ✅ React Router (Navigation)
- ✅ useState & useEffect (State Management)
- ✅ Props & Event Handling
- ✅ Forms Handling
- ✅ Responsive Design (Mobile, Tablet, Desktop)
- ✅ Loading States & Error Handling
- ✅ Modern UI with Tailwind CSS

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/maikel-479/tixora.git
cd tixora

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
